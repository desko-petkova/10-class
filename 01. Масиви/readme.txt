## Масив е:
	Едномерният масив е структура от данни, 
	която съхранява последователност от елементи от един и същи тип, 
	достъпвани чрез индекс.

## Създаване на масив
1. Деклариране и създаване

	int[] numbers = new int[5];  //Създава масив от 5 елемента (инициализирани по подразбиране).

2. Деклариране и инициализация

	int[] numbers = { 1, 2, 3, 4, 5 };

## Достъп до елементи
Индексите започват от 0.

	int x = numbers[0];  // първи елемент
	numbers[2] = 50;     // присвояване на стойност

## Дължина на масив

	int len = numbers.Length;

## Обхождане на масив
Обикновен for цикъл:

	for (int i = 0; i < numbers.Length; i++)
	{
    		Console.WriteLine(numbers[i]);
	}

foreach цикъл:

	foreach (int n in numbers)
	{
    		Console.WriteLine(n);
	}

## Типове данни
Масивите могат да са от всеки тип:

	string[] names = new string[] {"Ivan", "Dragan", "Ema", "Dora" };
	double[] prices = { 1.2, 3.4, 5.6 };
	bool[] flags = new bool[2];

## Характеристики
- Едномерният масив е статична структура – размерът му се задава при създаване.
- Елементите са съседни в паметта.
- Бърз достъп до елемент чрез индекс.
- Може да се комбинира с цикли и алгоритми.
